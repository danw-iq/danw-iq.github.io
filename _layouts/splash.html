What is iQmetrix Platform?

iQmetrix Platform is built from the ground up to support omnichannel retail experiences. Our developmental philosophy considers the operational complexity retailers face every day. Whether handling in-store, e-commerce or mobile transactions, we want to provide seamless experiences to retail staff and customers.

All our Platform services are designed with a publicly-facing REST API as a first-class citizen. This is the same API that our own developers use to build various products.
