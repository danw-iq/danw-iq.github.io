Authentication Tutorial (C#)

This tutorial is intended to walk you through the process of Obtaining an Access Token.

Start Tutorial

Prerequisites

Before starting this tutorial, ensure you have the following installed on your computer:

  • Visual Studio 2012 or newer
  • RestSharp
  • NuGet

    • Onboarding Package

    As part of the onboarding process, you will have received an onboarding package from the iQmetrix API team. This package provides you credentials and access details in order to perform the topics covered in this guide.

    Should you require information beyond the scope of this guide, or did not receive the onboarding package, contact Support.

    For the purposes of this guide, we will use the onboarding package to the right.

    Getting Started

    First, open Visual Studio and create a new project. For this tutorial, I suggest creating an empty Console Application.

    Open Program.cs in the sidebar and replace the contents of the file with the code to the right.

    First, import RestSharp:

  • Click Tools -> NuGet Package Manager -> Manage NuGet Packages for Solution
  • Click Browse
  • Type "Rest" in the search and press Enter
  • Click on RestSharp
  • A new Menu will appear to the right, click the Checkbox next to your project name
  • Click Install
  • If a Preview window appears, click OK

    • Next, import Newtonsoft.Json:

  • Click Tools -> NuGet Package Manager -> Manage NuGet Packages for Solution
  • Click Browse
  • Type "Newton" in the search and press Enter
  • Click on Newtonsoft.Json
  • A new Menu will appear to the right, click the Checkbox next to your project name
  • Click Install
  • If a Preview window appears, click OK

    • Filling In Values

    Next, use the values from your onboarding package to fill in the parameters highlighted to the right.

    Main

    First, the Main function calls ObtainingAnAccessToken. Later, it will output the response to the console.

    ObtainingAnAccessToken

    This is where the action happens - a POST request is performed on the URL https://accountsdemo.iqmetrix.net/v1/oauth2 using the Content-Type application/x-www-form-urlencoded.

    The result is raw JSON.

    OAuthToken

    The OAuthToken class is used to serialize the JSON response into an object using Newtonsoft.

    Performing The Request

    Run the program by pressing the Start button.

    A console window should appear and then quickly close.

    To view the Access Token, put a breakpoint at line 21.

    If you encounter an error, put a breakpoint at line 41 to see the error message.
