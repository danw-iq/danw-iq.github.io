Authentication Tutorial (C#)

This tutorial will show you how to get an Access Token.

You may be interested in this tutorial if you are a developer who is...
  • Integrating with iQmetrix APIs
  • Familar with C#
  • Familar with Visual Studio
Start Tutorial

Prerequisites

Before starting this tutorial, ensure you have the following installed on your computer:

  • Visual Studio 2012 or newer
  • RestSharp
  • NuGet

    • Onboarding Package

    As part of the onboarding process, you will have received an onboarding package from the iQmetrix API team.

    This package provides you credentials and access details in order to perform the topics covered in this guide.

    Should you require information beyond the scope of this guide, or did not receive the onboarding package, contact Support.

    For the purposes of this guide, we will use the onboarding package to the right.

    Getting Started

    First, open Visual Studio and create a new project. For this tutorial, it will be a Console Application.

    Open Program.cs and replace the contents of the file with the code to the right.

    Before running the application, RestSharp and Newtonsoft.Json must be installed.

    Installing RestSharp

    1. Click Tools -> NuGet Package Manager -> Manage NuGet Packages for Solution
    2. Click Browse above the search bar
    3. Type "Rest" in the search bar and press Enter
    4. Click on RestSharp
    5. A menu will appear to the right, ensure the check box next to your project name is selected and click Install
    6. If a Preview window appears, click OK
    7. Click Install

    Note: Instructions are for Visual Studio 2015, steps may differ for different versions.

    Installing Newtonsoft.Json:

    1. If you have NuGet Package Manager open, skip to step 4
    2. Click Tools -> NuGet Package Manager -> Manage NuGet Packages for Solution
    3. Click Browse above the search bar
    4. Type "Newton" in the search bar and press Enter
    5. Click on Newtonsoft.Json
    6. A menu will appear to the right, ensure the check box next to your project name is selected and click Install
    7. Click Install
    8. If a Preview window appears, click OK

    Note: Instructions are for Visual Studio 2015, steps may differ for different versions.

    Filling In Values

    Use the values from your onboarding package to fill in the parameters highlighted to the right.

    If you do not have an onboarding package, contact Support.

    How it Works

    Before running the request, take a look at each function to understand how it works.

    First, the Main function calls ObtainingAnAccessToken. Later, it will output the response to the console.

    ObtainingAnAccessToken

    This is where the action happens - a POST request is performed on the URL https://accountsdemo.iqmetrix.net/v1/oauth2 using the Content-Type application/x-www-form-urlencoded.

    The result is raw JSON.

    OAuthToken

    The OAuthToken class is used to serialize the JSON response into an object using Newtonsoft.

    Performing The Request

    Run the program by pressing the Start button.

    A console window should appear and then quickly close.

    To view the Access Token, put a breakpoint at line 21.

    If you encounter an error, put a breakpoint at line 41 to see the error message.
